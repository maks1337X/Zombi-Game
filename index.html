🌐 Об онлайн-режиме
Для полноценного онлайна нужен WebSocket-сервер. Рекомендуем бесплатные варианты:
• Partykit.io — бесплатный multiplayer-хостинг для игр
• Colyseus — open-source сервер для игр, есть бесплатный tier
• Ably / Pusher — realtime-каналы, бесплатно до 200 соединений

Схема работы: каждый игрок заходит по коду комнаты (как в Among Us), сервер синхронизирует позиции, выстрелы и состояние игры. Это доработка следующей версии!
